# My .NET React App


## Overview

Basic Todo app

## Installation


### Prerequisites

- .NET SDK [version]
- Node.js [version]
- npm

### Installation Steps

1. Clone the repository:

2. Navigate to the project directory:

    ```
    cd /Frontend
    ```

3. Install dependencies:

    ```
    npm install
    ```

4. Build the .NET project:

    ```
    dotnet build
    ```

5. Start the application:

    ```
    npm start
    dotnet run
    ```
